## âš™ï¸ Getting Started

### Prerequisites

- Node.js (v14+ recommended)
- npm or Yarn

### Setup & Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/myles-reid/ecommerce.git
   cd ecommerce
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
   _(or `yarn install`)_
3. Start the development server:
   ```bash
   npm start
   ```
4. Open your browser:
   ```
   http://localhost:3000
   ```
